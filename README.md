# LiteRP

Welcome to LiteRP! This project provides a **simple, lightweight AI roleplaying experience** built entirely in C#/.NET.

Tired of complex setups, large downloads (~500MB+), and managing Python/Node.js dependencies just to chat with an AI character? LiteRP aims to be different. Our goal is a **minimal, easy-to-run application** (ideally a single executable eventually) that lets you quickly load character cards and interact with them using local LLMs.

**Core Features (Current & Planned):**

*   Chat interface for AI roleplaying
*   Load characters from standard Character Cards (TavernAI format - Planned)
*   Connect to LLM backends (currently supports **Ollama**)
*   Lightweight and minimal dependencies
*   Cross-platform (where .NET runs)
